# Metadata Registry

This repository contains submodules so you will need to execute:

```
git clone --recursive https://github.com/zmhassan/metadatapoc
```

or

```
git clone https://github.com/zmhassan/metadatapoc
git sub module init
```



## Building

```
mvn clean install
cd assembly/target
tar -zxvf <NAME>.tar.gz
```
